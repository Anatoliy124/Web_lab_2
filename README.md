# To-Do List - Веб-приложение

Веб-приложение для управления задачами с возможностью создания, редактирования, удаления, фильтрации, сортировки и поиска задач.

## Технологии

- Чистый JavaScript (ES6+)
- HTML5
- CSS3
- DOM API
- LocalStorage API

## Функциональность

- ✅ Добавление новых задач с названием и датой
- ✅ Удаление задач
- ✅ Редактирование задач (текст и дата)
- ✅ Отметка задач как выполненных
- ✅ Сортировка задач по дате (возрастание/убывание)
- ✅ Фильтрация задач по статусу (все/активные/выполненные)
- ✅ Поиск задач по названию
- ✅ Сохранение задач в localStorage
- ✅ Изменение порядка задач с помощью drag-and-drop

## Как запустить

### Способ 1: Открыть файл напрямую

Просто откройте файл `index.html` в браузере:
- Двойной клик по файлу `index.html`
- Или правой кнопкой мыши → "Открыть с помощью" → выберите браузер

### Способ 2: Использовать локальный сервер (рекомендуется)

#### С Python:
```bash
# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Затем откройте в браузере: `http://localhost:8000`

#### С Node.js (http-server):
```bash
# Установка (если не установлен)
npm install -g http-server

# Запуск
http-server -p 8000
```

Затем откройте в браузере: `http://localhost:8000`

#### С PHP:
```bash
php -S localhost:8000
```

Затем откройте в браузере: `http://localhost:8000`

## Использование

1. **Добавление задачи**: Введите название задачи в поле ввода, при необходимости выберите дату и нажмите "Добавить"

2. **Отметка как выполненной**: Установите флажок слева от задачи

3. **Редактирование**: Нажмите кнопку "Редактировать" на задаче

4. **Удаление**: Нажмите кнопку "Удалить" на задаче

5. **Поиск**: Введите текст в поле поиска для фильтрации задач по названию

6. **Фильтрация**: Выберите фильтр из выпадающего списка (Все/Активные/Выполненные)

7. **Сортировка**: Выберите способ сортировки из выпадающего списка

8. **Изменение порядка**: Перетащите задачу мышью в нужное место списка

## Требования

- Современный браузер с поддержкой ES6+ (Chrome, Firefox, Safari, Edge)
- JavaScript должен быть включен
